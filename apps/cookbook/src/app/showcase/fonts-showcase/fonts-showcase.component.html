Out of the box Kirby will style text with a set of predefined font sizes and line heights. These styles are globally applied to Paragraph elements (<p>) and HTML Section Headings (<h1> to <h4>) when including Kirby global styles as described in our readme

Font size and corresponding line height can also be set by applying the font CSS classes, e.g. kirby-text-large.

Finally the font-size and line-height SCSS utility functions can be used when writing your own CSS rules.
Note: When setting the font size this way ensure the corresponding line height is set as well (and optionally the font weight). 

@use '@kirbydesign/designsystem/scss/utils' as *;

.my-own-class {{ '{' }}
  font-size: font-size("l");
  line-height: line-height("l");
  ...
{{ '}' }}

Display 1

font-size: font-size("xxxxl"); font-weight: font-weight("black"); line-height: normal; class="kirby-text-display-1"

Display 2

font-size: font-size("xxxl"); font-weight: font-weight("black"); line-height: normal; class="kirby-text-display-2"

Display 3

font-size: font-size("xxl"); font-weight: font-weight("black"); line-height: normal; class="kirby-text-display-3"

h1 - Heading 1

font-size: font-size("xl"); font-weight: font-weight("black"); line-height: line-height('xl'); class="kirby-text-xlarge"

h2 - Heading 2

font-size: font-size("l"); font-weight: font-weight("bold"); line-height: line-height('l'); class="kirby-text-large"

h3 - Heading 3

font-size: font-size("m"); font-weight: font-weight("bold"); line-height: line-height('m'); class="kirby-text-medium"

h4 - Section header

font-size: font-size("n"); font-weight: font-weight("bold"); line-height: line-height('n'); class="kirby-text-normal-bold"

Bold text

font-weight: font-weight("bold"); class="kirby-text-bold"

p - Paragraph (default body font)

font-size: font-size("n"); line-height: line-height('n'); class="kirby-text-normal"

Small text

font-size: font-size("s"); line-height: line-height('s'); class="kirby-text-small"

Small text - light

font-size: font-size("s"); font-weight: font-weight("light"); line-height: line-height('s'); class="kirby-text-small-light"

x-small text

font-size: font-size("xs"); line-height: line-height('xs'); class="kirby-text-xsmall"

xx-small text

font-size: font-size("xxs"); line-height: line-height('xxs'); class="kirby-text-xxsmall"
