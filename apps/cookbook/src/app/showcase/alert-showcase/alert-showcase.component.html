Accessibility

Alerts have a dialog role and implement focus management and keyboard support accordingly.

When opened, focus is moved into the alert. Focus is trapped within the alert while it is open, preventing users from navigating to elements outside the alert. Upon closing, focus is moved back to where it was before showing the alert.

Labeling

The alert title is automatically used as a label to add context for assistive technologies. For example, screen readers use the label to announce when the modal dialog appears and to give proper context when interacting with the content within.

This emphasizes the importance of providing a meaningful title for all alerts.
