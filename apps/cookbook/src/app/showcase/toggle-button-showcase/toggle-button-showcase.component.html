Toggle buttons allow users to switch between two states, such as on/off or activated/deactivated. They are commonly used in settings and preferences.

Icon Only & Attention Level

A toggle button wraps two kirby-buttons, one marked with the checked attribute and one with unchecked . The [checked]="true" property of the toggle button element itself can be used to set the initial state of the toggle button.

Disabled

Each of the nested buttons can be disabled and will then not toggle the state.

In the example above aria-disabled is used in place of the disabled attribute to ensure that the element is still focusable by keyboard and discoverable by assistive technology. Read more about accessible disabled buttons in the button docs .

Theme Color

In contrast to regular buttons, buttons inside a toggle button accept the themeColor property for using the success, warning, and danger color palette.

Angular Forms

The toggle button can be used directly in template driven and reactive forms . To use toggle button in a reactive form, bind your form control to <kirby-toggle-button> with simple data binding:

<kirby-toggle-button [formControl]="yourFormControl"> // or simply: <kirby-toggle-button formControlName="yourFormControlName">
